Security Considerations

Security is important for all topics regarding websites, as web applications usually store personal data within them and are accessible throughout the world, and as such are prone to targeting from hackers. In consideration toward the security of my website, I would recommend a firewall be put in place as a
single layer perimeter defence to reduce the risk of attacks on the database server as well as the web server.
This is important for both security as well as for data protection and legal reasons. The General Data Protection Regulation (GDPR), a regulation brought
in to protect the private data of users would have to be considered for the website, as breaches of this are subject to penalties.
With this in mind, the user has a requirement to guarantee the safety of users data, as well users having a right for this data to be expunged upon request (Office, 2018).

Another consideration I would put in place to enhance the security of the website would be the implementation of a user login for the admin page. The issue of the admin page is that private data is publicly available in its current situation, this also breaches GDPR regulations (Office, 2018).
